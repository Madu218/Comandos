## Listas:
> Estrutura de dados que armazena uma coleção ordenada de itens.
* len(lista)
* sum(lista)
* min(lista)
* max(lista)
* lista.count(valor)
* lista.count(valor, idx_inicio)
* lista.index(valor)
* lista.index(valor, idx_inicio)
* lista.sort()
* lista.sort(reverse=True)
* sorted(lista)
* lista.insert(idx)
* lista.remove(valor)
* lista.pop(idx)
* lista += [valor]
* lista[idx] = valor
